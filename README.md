# 🌙 AstroIndia — Vedic Astrology Platform

A complete Vedic astrology web app built as a single static HTML file. No frameworks, no dependencies, no build step.

**Live:** https://sreenathsjk.github.io/astroindai/

## Features

- **Kundli** — Full birth chart, planetary positions, Vimshottari Dasha, doshas & yogas
- **Daily Horoscope** — All 12 rashis with career, love, health & finance scores
- **Kundli Matching** — Ashtakoota Guna Milan compatibility calculator
- **Panchang** — Tithi, Nakshatra, Yoga, Karana, Vara & Muhurat for any date
- **Numerology** — Life path, destiny, soul urge & personality numbers
- **AI Astrologer** — Live chat powered by Claude AI (Jyotish AI)

## Tech Stack

- Pure HTML, CSS, JavaScript (zero dependencies)
- Mathematical planetary calculations (Lahiri Ayanamsa)
- Claude API for AI chat
- Hosted on GitHub Pages

## Deployment

Just upload `index.html` to any static host — GitHub Pages, Netlify, or Vercel.
